# Analizador de Hábitos

Proyecto en Python para registrar y analizar hábitos diarios.

## Funciones
- Agregar registros diarios
- Ver registros
- Ver resumen estadístico
- Graficar estado de ánimo

## Tecnologías usadas
- Python
- Pandas
- Matplotlib

## Objetivo
Practicar análisis de datos y visualización con información personal.
